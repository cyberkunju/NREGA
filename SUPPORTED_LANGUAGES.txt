# MGNREGA Report Card - Supported Languages

## Language Coverage (15 Languages)

1. English (en) - Primary language, default
2. Hindi (hi) - Official language of India
3. Tamil (ta) - Tamil Nadu, Puducherry
4. Telugu (te) - Andhra Pradesh, Telangana
5. Malayalam (ml) - Kerala, Lakshadweep
6. Kannada (kn) - Karnataka
7. Marathi (mr) - Maharashtra, Goa
8. Gujarati (gu) - Gujarat, Dadra and Nagar Haveli, Daman and Diu
9. Punjabi (pa) - Punjab, Chandigarh
10. Odia (or) - Odisha
11. Assamese (as) - Assam
12. Mizo/Lushai (lus) - Mizoram
13. Manipuri (mni) - Manipur
14. Nepali (ne) - Sikkim
15. Kokborok (kok) - Tripura

## Coverage Analysis

### Population Coverage
- These 15 languages cover ~95% of India's population
- Includes all 22 scheduled languages with significant MGNREGA participation

### Geographic Coverage
- All major states and UTs represented
- Aligns with states having highest MGNREGA implementation
- Regional language matches state-level data access patterns

### Implementation Priority

**Tier 1 (High Priority)**: Launch Phase
- English (en) - Universal access
- Hindi (hi) - 40%+ population, pan-India
- Tamil (ta) - High MGNREGA participation state
- Telugu (te) - High MGNREGA participation state
- Marathi (mr) - Maharashtra has significant rural workforce

**Tier 2 (Medium Priority)**: Post-Launch
- Malayalam (ml)
- Kannada (kn)
- Gujarati (gu)
- Punjabi (pa)
- Odia (or)

**Tier 3 (Lower Priority)**: Based on Demand
- Assamese (as)
- Mizo/Lushai (lus)
- Manipuri (mni)
- Nepali (ne)
- Kokborok (kok)

## Technical Considerations

### Bundle Size Impact
- 15 languages × ~500 translation keys = ~7,500 strings
- Estimated size: 150-300 KB per language (uncompressed)
- Total: ~2.25-4.5 MB for all languages
- With lazy loading: Only 150-300 KB loaded per user session

### Translation Maintenance
- 15 languages require coordinated updates for new features
- Need translation management system (TMS) or workflow
- Consider professional translation services for accuracy
- Government terminology must be precise

### Font Support
- Devanagari: Hindi, Marathi, Nepali, Kokborok
- Tamil script: Tamil
- Telugu script: Telugu
- Malayalam script: Malayalam
- Kannada script: Kannada
- Gujarati script: Gujarati
- Gurmukhi script: Punjabi
- Odia script: Odia
- Bengali-Assamese script: Assamese
- Mizo uses Latin script
- Manipuri uses Bengali script (Meitei Mayek also exists)

Recommended: Use Noto Sans family (Google Fonts) for comprehensive coverage

### Right-to-Left (RTL) Support
- Not required for these languages (all LTR)
- Simplifies CSS and layout implementation

## Recommendations

1. **Phased Rollout**: Start with Tier 1 (5 languages), expand based on usage analytics
2. **Lazy Loading**: Essential - load only selected language
3. **Translation Management**: Use Crowdin, Lokalise, or similar for workflow
4. **Quality Assurance**: Native speakers for each language to review translations
5. **Fallback Strategy**: English as fallback for missing translations
6. **Analytics**: Track language selection to prioritize future additions
7. **Community Contribution**: Consider allowing community translations for Tier 3 languages

## Estimated Effort

- **Setup Infrastructure**: 2-3 days (i18next, language switcher, font loading)
- **Translation per Language**: 3-5 days (500+ strings, review, QA)
- **Total for 15 Languages**: 45-75 days of translation work
- **Maintenance**: Ongoing - every new feature needs 15 translations

## Cost Considerations

- Professional translation: ₹1-3 per word
- ~5,000 words per language
- ₹5,000-15,000 per language
- Total: ₹75,000-2,25,000 for all 15 languages

Alternative: Government translation services or volunteer translators from respective states
